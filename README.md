# S.C.O.P.E. FeedForward Coach

Constitutional AI coaching bot using Danny Simms' S.C.O.P.E. FeedForward Model.

## Overview
A voice-powered AI coach that provides growth-oriented feedback using the S.C.O.P.E. framework:
- **S**ituation Awareness
- **C**hoices Recognition
- **O**utcomes Visualization
- **P**urpose Alignment
- **E**ngagement Commitment

## Constitutional AI Principles (Yamas)
- Ahimsa (non-violence): No harmful or controlling language
- Satya (truth): Evidence-based coaching principles
- Asteya (non-stealing): Respect autonomy and agency
- Brahmacharya (moderation): Psychological safety
- Aparigraha (non-attachment): Future-focused, not blame

## Architecture
- Frontend: Vite + React + TypeScript
- Backend: FastAPI + Python
- Voice: OpenAI Realtime API
- Inference: Groq Sidecar for parallel analysis

## Getting Started
1. Clone repository
2. Install dependencies: `npm install` (frontend), `pip install -r requirements.txt` (backend)
3. Set environment variables in `.env`
4. Run: `npm run dev` (frontend), `uvicorn main:app` (backend)
